# **App Name**: Template Genesis

## Core Features:

- Base Template Setup: Initializes a Next.js project with Tailwind CSS.
- Shadcn UI Integration: Integrates Shadcn UI components to provide pre-designed UI elements.
- Radix UI Configuration: Sets up Radix UI primitives for accessibility and advanced UI patterns.
- Global CSS Baseline: Establishes a global CSS baseline for consistent styling across the application using a `global.css` file.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) for a modern and calming feel.
- Background color: Light indigo (#E8EAF6), subtly desaturated, complementing the primary.
- Accent color: Violet (#7986CB) to highlight interactive elements.
- Body and headline font: 'Inter', a sans-serif providing a modern and neutral aesthetic.
- Use a clean, minimalist layout to showcase content effectively.
- Incorporate simple, modern icons from a set like Feather or Lucide.
